step 1: Create a directory
        mkdir color-palette-extractor

step 2: open this directory in visual studio code

step 3: Create a virtual environment
        python3 -m venv venv

step 4: Activate your virtual environment
        source venv/bin/activate

step 5: (Optional but good idea) Upgrade pip
        pip install --upgrade pip

step 6: install requires libraries
        pip install colorgram.py pillow matplotlib

step 7: Copy any image (e.g. image1.jpg) into the color-palette-extractor directory.

step 8: Create a file named e.g. extract_palette.py in the project folder
        touch extract_palette.py

step 9: Write pythoon code in extract_palette.py

step 10: Run Your Script
         python extract_palette.py

step 11: Want to save the palette bar? Add this before plt.show()
         plt.savefig("palette.png")

step 12: Deactivate Your Virtual Environment when you're done
         deactivate
